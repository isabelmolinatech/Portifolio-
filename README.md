# ✨ Portfólio Pessoal

Bem-vinda(o) ao meu projeto de portfólio pessoal!  
Este site foi desenvolvido com o objetivo de apresentar meu perfil e colocar em prática minhas habilidades em HTML e CSS.

---

## 📌 Sobre o projeto

Este portfólio foi criado para praticar HTML e CSS e reunir, em um só lugar, informações importantes sobre mim na área de tecnologia.  
A proposta do projeto é transmitir minha identidade de forma visual, profissional e acessível.

---

## 🎯 Objetivo

Desenvolver uma página pessoal para:

- apresentar meu perfil acadêmico
- destacar minhas habilidades
- compartilhar meus links profissionais
- construir uma presença digital mais organizada

---

## 🚀 Tecnologias utilizadas

- HTML
- CSS

---

## 💻 Estrutura do site

O projeto contém:

- nome em destaque
- apresentação acadêmica
- descrição pessoal
- botões com acesso ao GitHub e ao LinkedIn
- foto de perfil
- identidade visual com tema escuro e detalhes em roxo

---

## 🎨 Design

O site foi desenvolvido com uma proposta visual:

- simples
- moderna
- elegante
- organizada

A paleta escura com detalhes em roxo foi escolhida para transmitir personalidade, contraste e uma estética mais tecnológica.
